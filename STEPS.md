# Project Steps

1. Prepared Excel file with Windows Server CI data.
2. Opened ServiceNow Developer Instance.
3. Navigated to System Import Sets > Load Data.
4. Created a new import set table.
5. Uploaded the Excel file.
6. Created a Transform Map.
7. Selected target table: Windows Server [cmdb_ci_win_server].
8. Used Mapping Assist to map source columns to target fields.
9. Ran the transform.
10. Verified successful import processor result.
11. Checked records in the Windows Servers table.
12. Reviewed CI relationships in the CI Relationships table.
